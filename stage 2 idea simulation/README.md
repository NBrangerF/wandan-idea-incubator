
# 完蛋！我被好想法包围了！ 🧪🧠

> **"在动手之前，先在模拟中预演成功。"**

本作是一个 30 分钟一次性通关的“思维压力测试（Simulation）”：它通过 **逆向溯源 (Backcasting)** 和 **AI 逻辑压力测试**，在项目启动前帮助你构建严密的行动地图。

## 🌀 模拟核心 (The Core Mechanics)

应用由一个支持型“思维教练”引导，通过以下机制确保你的逻辑无懈可击：

1.  **BUILD (逻辑构建)**：从终点 (Goal) 出发逆向拆解。不谈细节，只谈结构。
2.  **TEST (压力测试)**：AI 识别逻辑裂痕，通过“微验证任务”确保每一个 Done 都是可验收的。
3.  **ENTRY POINT (起点识别)**：模拟的终点是找到“下一分钟”的具体起点。当系统识别到 `START ->` 节点时，意味着你的思维已具象化为可执行动作。

## 🛠️ 最新特性 (V3.0 Enhancements)

- **全量地图导出 (Pro Export)**：支持一键导出 PNG (2X 高清) 或动态 PDF。不再受视窗限制，自动捕获所有逻辑节点。
- **动态逻辑筋络 (Tactile Connections)**：
  - **流动动画**：线条带有虚线滚动效果，直观显示因果流向。
  - **语义配色**：PREREQ (前置)、INPUT (输入)、VALIDATION (验证) 拥有专属配色与永久标签。
  - **视觉聚焦**：选中节点时，相关路径会高亮，无关背景自动淡化。
- **车道布局系统**：
  - **L1 战略层**：定义目标与范围。
  - **L2 战术层**：构建执行步骤。
  - **L3 运行层**：质量检查与风险验证。
- **Gemini 3 Pro 驱动**：利用最新的思维链模型进行深度推理，捕捉你思维中的模糊地带。

## 🚀 快速启动

1.  **环境配置**：
    确保环境中有 `process.env.API_KEY`。
2.  **启动模拟**：
    输入你的项目目标，教练将引导你从右向左构建逻辑骨架。
3.  **锁定起点**：
    当看到带 🚀 标识的起点卡片时，点击“导出”即可获得完整的项目行动地图。

## ⚠️ 系统要求

- Node.js 18+
- 支持 `gemini-3-pro-preview` 或 `gemini-3-flash-preview` 模型的 Google Gemini API Key。
